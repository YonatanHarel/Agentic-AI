# CrewAI projects

Agentic ai projects using CrewAI

## How to create new CrewAI project
In orderto create new CrewAI project we need to run the following command:
```bash
crewai create crew <project_name>
```

We can also create crewAI 'flow' project by running:
```bash
crewai create flow <project_name>
```
This will create workflow project
